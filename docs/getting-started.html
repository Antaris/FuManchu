Getting Started with FuManchu
An introduction into getting your project up and running with FuManchu

Installing FuManchu

FuManchu is available via the NuGet.org public feed, which you can easily install via the package manager, or typing the following in the package manager host:

Install-Package FuManchu

Referencing and using FuManchu

First things first, after installing the package (see above), you need to import the FuManchu namespace into your code file

        
// Import the FuManchu namespace.
using FuManchu;

Compiling your first template

FuManchu provides the static Handlebars class which itself provides a default instance of the IHandlebarsService exposed as a singleton instance.

        
string template = "Hello ";

// Compile your template.
var templateFunc = Handlebars.Compile("name", template);
// Execute your template with your model.
string result = templateFunc(new { name = "Matt" });
